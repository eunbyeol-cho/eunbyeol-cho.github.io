# Eunbyeol Cho - Personal Website

This is my personal academic website hosted on GitHub Pages.

## Theme

This website is based on the [Long Haul](https://github.com/brianmaierjr/long-haul) Jekyll theme by [Brian Maier Jr.](https://github.com/brianmaierjr). Long Haul is a minimal Jekyll theme built with SASS / GULP / BROWSERSYNC / AUTOPREFIXER and focuses on long form blog posts.

The theme is licensed under the [MIT License](LICENSE), which allows free use without attribution, but I'm acknowledging the original creator here as a courtesy.

## Original Theme Features

- Minimal, Type Focused Design
- Built with GULP + SASS + BROWSERSYNC + AUTOPREFIXER
- SVG Social Icons
- Responsive Nav Menu
- XML Feed for RSS Readers
- Dark Mode support via prefers-color-scheme
